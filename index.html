Basics

git init

Create an empty Git repository in the specified directory

git init <directory-name>

Use with no arguments to initialize in the current directory

git init

git clone

Clone repository from specified location to local machine, via HTTP or SSH

git clone <repo>

git status

List files that are staged, unstaged and untracked

git status

git log

Display the commit history (default format)

git log

git add

Stage all changes in the directory for the next commit - directory can be replaced by a single file, or use a full stop for all

git add <directory>
git add <file>
git add .

git remote

Create a new connection to a remote repository

git remote add <name> <url>

git pull

Fetch the remote copy of the current branch and merge into local copy

git pull

Branches

git branch

List all of the branches in repository.

git branch

Add a branch arg to create a new branch with the name specified

git branch <branch-name>

Create a new branch and switch to it

git checkout -b <branch-name>

git merge

Merge specified branch into the current branch

git merge <branch-name>

git branch -d

Delete a branch

git branch -d <branch-name>

Commit

git commit

Commit the staged snapshot - entering text edit to add a message

git commit

Commit the staged snapshot, using the specified message as the commit message

git commit -m "message"

Replace the last commit with staged changes and last commit combined.

git commit --amend

git push

Push the branch to the remote repository, along with commits and objects

git push

Config

git config

Open the global config file in a text editor

git config --global --edit

Set the author name to use is commits for the current user

git config --global user.name <name>

Set the author email to use is commits for the current user

git config --global user.email <email>

History

git rebase

Apply any commits of the current branch ahead of the specified one

git rebase <branch>

git reset

Clear the staging area, rewiting the working tree from a specific commit

git reset --hard <commit>

git reflog

Show a log of changes to the local repository's HEAD/p> git reflog

Undo

git reset

Revert all changes to the last commit

git reset --hard

git clean

Remove new and untracked files and directories

git clean -f

To preview the files that will be permanently deleted

git clean --dry-run

git revert

Revert a commit in a new commit

git revert <commit-id>
